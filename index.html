PetWalk 개인정보 처리방침

시행일: 2026년 2월 27일

1. 수집하는 정보

PetWalk는 다음 정보를 기기 내(localStorage)에만 저장하며, 외부 서버로 전송하지 않습니다:

- 반려동물 프로필 정보 (이름, 품종, 생년월일, 성별, 체중, 사진)
- 산책 기록 (경로, 시간, 거리, 활동 로그)
- 위치 정보 (산책 추적 시에만 사용)

2. 위치 정보

GPS 위치 정보는 산책 경로 추적 목적으로만 사용되며, 산책 종료 후 기기 내에만 저장됩니다. 외부 서버로 전송되지 않습니다.

3. 카메라/사진 접근

카메라 및 사진 라이브러리 접근은 반려동물 프로필 사진 등록 및 산책 중 사진 촬영을 위해 사용됩니다.

4. 데이터 저장

모든 데이터는 사용자의 기기에만 저장됩니다. 별도의 서버를 운영하지 않으며, 사용자 데이터를 수집하거나 제3자와 공유하지 않습니다.

5. 날씨 정보

현재 위치 기반 날씨 정보를 표시하기 위해 Open-Meteo API를 사용합니다. 이때 위도/경도 좌표만 전송되며 개인 식별 정보는 포함되지 않습니다.

6. 데이터 삭제

앱 삭제 시 모든 데이터가 자동으로 삭제됩니다. 브라우저 사용 시 localStorage를 초기화하면 모든 데이터가 삭제됩니다.

7. 문의

개인정보 처리방침에 대한 문의사항이 있으시면 앱 스토어 페이지를 통해 연락해주세요.
